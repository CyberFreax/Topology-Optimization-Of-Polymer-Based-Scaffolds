📌 Project: Hybrid Octet–Voronoi Lattice Scaffold

This project focuses on the computational design and optimization of a hybrid bio-inspired lattice structure for potential biomedical implant applications.

The core idea is to combine:

A periodic octet-truss lattice (core) for predictable load-bearing behavior
A stochastic Voronoi lattice (outer shell) to mimic natural trabecular bone and enhance osseointegration
🚧 Current Status: Ongoing

This is an active research and development project, where the objective is to build a complete pipeline from parametric design → geometry processing → finite element validation → physical testing.

⚙️ What the Project Covers
Parametric Lattice Generation
Grasshopper-based workflow for generating 3D Voronoi structures
Control over point spacing, strut thickness, and porosity (~60%)
Standardized Geometry Design
Models constrained to ASTM D695 compression testing dimensions
Domain: 25.4 × 12.7 × 12.7 mm
Hybrid Topology Integration
Merging structured octet-truss core with irregular Voronoi outer topology
Aim: reduce stress concentrations and improve load distribution
FEA & Meshing Challenges
Handling complex lattice intersections in ANSYS
Using patch-conforming tetrahedral meshing to overcome failures
Material Modeling
PLA-based scaffold simulation
Evaluating stress distribution and mechanical response
🎯 Objective

To develop a mechanically optimized, bioresorbable polymer scaffold that:

Reduces stress shielding
Mimics natural bone architecture
Performs reliably under compression, impact, and cyclic loading
🔬 Future Work

Physical validation using UTM
AI/ML-based topology optimization
